# Improved Prompt

You are generating occasion-specific text anchors for a multimodal fashion recommender that uses text embeddings (e.g., CLIP). For each occasion listed, produce two outputs: a conceptual description and an embedding prompt set.

Occasions (process in this exact order):
- wedding_guest
- party_night_out
- work_office
- formal_event
- casual_everyday
- date_night
- winter_outerwear
- smart_casual

STEP 1 — Conceptual Description
Write ONE paragraph (4–6 sentences) that defines clothing appropriate for the occasion. The paragraph must:
- State the formality level explicitly.
- Mention typical silhouettes (e.g., fitted dress, tailored blazer, relaxed knits).
- Mention fabric types or material qualities (e.g., wool suiting, satin, denim, quilted).
- Mention color tendencies or palette (e.g., muted neutrals, jewel tones, high-contrast).
- Mention structural or styling cues (e.g., layering, clean lines, embellishment, coverage).
- Use neutral descriptive language (no marketing tone).
- Avoid generic wording and avoid overlapping characteristics with other occasions.

STEP 2 — Embedding Prompt Set
Convert the paragraph into 5 short prompts suitable for embeddings. Each prompt must:
- Be 8–15 words.
- Focus on ONE semantic dimension (e.g., silhouette, fabric, formality, structure, social context).
- Use distinct wording across the five prompts.
- Be clearly differentiable from prompts of other occasions.
- Contain no brand names.

Global constraints:
- Make different occasions strongly separable in meaning; avoid shared phrases across occasions (e.g., “elegant outfit”) unless unavoidable.
- Keep content strictly about clothing attributes and context; do not add styling advice.

Output format (repeat per occasion):
Occasion: <name>
Conceptual Description: <paragraph>
Embedding Prompts:
- <short prompt>
- <short prompt>
- <short prompt>
- <short prompt>
- <short prompt>

## Alternative Approaches
1. Add a per-occasion "contrastive keywords" line that lists 3-5 distinguishing terms to enforce separability.
2. Split Step 2 prompts into fixed semantic slots: silhouette, fabric, formality, structure, social context.
3. Include a short "anti-features" line (what to avoid) per occasion to reduce overlap.

## Validation Checklist
- Each occasion has exactly one 4–6 sentence paragraph.
- Paragraph includes formality, silhouette, fabric, palette, and structural cues.
- Five prompts per occasion, each 8–15 words.
- Prompts are semantically distinct and do not share wording across occasions.
- No brand names or marketing language appear.
- Output format matches the required template for every occasion.

## Edge Cases
- Overlap between formal_event and wedding_guest language.
- winter_outerwear drifting into casual_everyday descriptions.
- smart_casual collapsing into work_office phrasing.
- party_night_out vs date_night ambiguity in tone and silhouettes.
